## 7. Manuscript Risk Warning

Because 2022-2024 observations are fitted/extrapolated rather than official observed data, the manuscript should not describe the whole 2006-2024 sample as fully observed official panel data.

Recommended presentation choice:

- Option A, more conservative: make 2006-2021 observed data the main analysis and present 2006-2024 fitted/extrapolated data as a sensitivity or extension analysis.
- Option B, acceptable only with explicit disclosure: use 2006-2024 as the main analysis, but clearly state in the abstract, data section, table notes, figure captions, and limitations that 2022-2024 are fitted/extrapolated values.

Preferred recommendation: Option A is lower risk for peer review because it avoids treating fitted data as equivalent to observed official data. If the paper keeps Option B, the data construction method and fitted-data indicator must be disclosed prominently.
